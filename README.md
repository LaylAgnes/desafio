# Desafio - Pagina de perfil responsiva

Este projeto é uma simples pagina de apresentação pessoal, feita em um desafio de 1h de frontend.
Ela contém:
- Uma foto de perfil.
- Nome completo.
- Biografia.
- Hobbies.
- Um layout responsivo


  # Tecnologias Utilizadas:
  - HTML5
  - CSS3


   # Funcionalidades:
    - Layout centralizado
    - Responsivo
